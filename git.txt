Para inicializar um diretório como um repositório Git e enviá-lo para o GitHub, você pode seguir os seguintes passos:

    Inicializar o diretório como um repositório Git:

git init

Este comando criará um novo diretório .git no seu diretório atual, que conterá todos os arquivos e configurações do repositório Git.

    Adicione todos os arquivos do diretório ao staging:

git add .

Este comando adicionará todos os arquivos do diretório atual ao staging, preparando-os para serem commitados.

    Crie um commit com uma mensagem de commit:

git commit -m "Initial commit"

Este comando criará um novo commit com a mensagem "Initial commit". Você pode substituir essa mensagem por uma descrição mais detalhada do que foi feito no commit.

    Crie uma conta no GitHub e crie um novo repositório:

https://github.com/username/repository-name.git
